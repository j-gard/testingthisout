Hello World! Let me introduce you to git!

code . is for opening this editor through your terminal.

git add yourfile.language (ex. testing.html) is for adding a file to the editor via git in your terminal.

git commmit -m "note what you changed here" is essentially your save command. note that you are saving the FOLDER and everything in it rather than each FILE individually.

git status is for making sure that your code is saved and tracked.

git push is for pushing your saved files online so you can edit them later or so others can edit them.
